# Content Analytics Demo

A live analytics report produced by the content tracking pipeline built into **[AILA Marketer](https://ailamarketing.com)** — an AI-powered content automation service.

## What This Is

The report (`index.html`) is a self-contained dashboard showing YouTube performance metrics for **LincolnPlaysRivals**, a gaming channel featuring father-and-son *Marvel Rivals* highlight reels. Every video on the channel is produced end-to-end by an AI content pipeline.

The dashboard tracks how that content performs — views, watch time, retention, and engagement — pulled directly from YouTube Studio and updated daily, with data stored historically so performance trends are visible over time.

## Why This Exists

This is a portfolio demo of the analytics component offered as part of [AILA Marketer](https://ailamarketing.com). The pipeline collects social media engagement metrics, builds a historical record, and renders a shareable report — giving clients a clear picture of how their content is performing over time.
